== Description ==

WordPress plugin that allows you to display the responsive WHMCS Domain Checker in a widget. Appearance settings and the list of suggested domains can be configured via WordPress admin.

== Installation ==

1. Upload the whmcs-domain-checker folder to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go the widgets section in WordPress and drag the WHMCS Domain Checker widget to your widget area.
4. Input the path to your whmcs directory (http://www.example.com/whmcs/) and adjust the appearance settings.
